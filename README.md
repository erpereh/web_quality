# Web Quality

Landing page desarrollada con Vite y React.

## Requisitos

- Node.js
- npm

## Instalacion

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El servidor de desarrollo se inicia en `127.0.0.1`.

## Build de produccion

```bash
npm run build
```

El build se genera en `dist/`. Esta carpeta no se sube al repositorio porque se puede regenerar desde el codigo fuente.

## Vista previa del build

```bash
npm run preview
```

## Estructura principal

- `src/`: codigo de la aplicacion.
- `public/`: assets publicos usados por Vite.
- `referencias/`: materiales de referencia del proyecto.
